## The most challenging part

The most challenging part of this assignment was using GitHub Spec-Kit — I had never used it before, I had only heard of it. But after finishing the assignment, I realised it is actually a very practical, work-saving workflow.

## How AI tools helped (or hindered)

AI tools sped up my workflow significantly. With **Claude Code** I went from spec to working code in one sitting, and the **Supabase, Playwright, and Context7 MCPs** let me hand off most of the work I would normally do manually — creating the Supabase project, applying the schema migration, checking the live database from the terminal. I barely had to open the Supabase dashboard.
